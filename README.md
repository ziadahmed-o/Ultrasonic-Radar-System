## 📐 System Specifications & Components

* **Main Controller:** **Arduino Uno** for processing sensor data and controlling mechanical movement.
* **Sensing Module:** **Ultrasonic Sensor (HC-SR04)** for distance measurement.
* **Actuation:** **Micro Servo Motor (SG90)** to rotate the sensor 180 degrees.
* **Visual Interface:** **Processing IDE** used to create the radar graphical interface.
